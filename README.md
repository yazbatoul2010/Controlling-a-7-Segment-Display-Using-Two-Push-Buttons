# Controlling-a-7-Segment-Display-Using-Two-Push-Buttons
In this project, an Arduino board is used to control a 7-segment display using two push buttons and an indicator LED, in order to implement a countdown timer system with a visual notification at the end. The first push button is used to select the number displayed on the 7-segment display. Each press increments the displayed value from 1 to 9. When the value reaches 9, it rolls over back to 1. The button state is read through a digital input pin, and software debouncing is applied to ensure stable and accurate input detection. The second push button is responsible for starting the countdown process from the previously selected number. Once pressed, the Arduino decreases the displayed number at a fixed time interval (for example, one second) until it reaches zero. Timing control is achieved using appropriate Arduino functions When the countdown finishes and the value reaches zero, the Arduino activates an indicator LED, which turns on to signal that the countdown has ended. The LED remains on until the system is reset for a new operation. When the user starts selecting a new number using the first push button, the system automatically turns off the indicator LED, indicating that the system has returned to the setup mode and is ready for a new countdown cycle.
